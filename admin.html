Alur Kerja:
  1. Data produk dari products.json otomatis dimuat saat halaman dibuka.
  2. Gunakan form untuk Tambah Produk Baru atau klik Edit pada daftar produk.
  3. Isi path gambar, preview akan muncul jika path benar.
  4. Setelah selesai menambah/mengedit/menghapus, klik Download products.json atau Copy JSON.
  5. Upload file products.json yang baru ke repository GitHub Anda (timpa file lama).
Pastikan path gambar benar dan JSON yang dihasilkan valid sebelum diupload. Gunakan JSONLint untuk validasi.
